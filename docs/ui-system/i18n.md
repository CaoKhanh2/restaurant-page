# 🌐 Đa ngôn ngữ (i18n) — FR / EN

**File:** `src/scripts/i18n.js` · init trong `main.js → initI18n()` · lang init inline trong `Layout.astro`

---

## Cơ chế

1. **Từ điển** `translations = { fr: {...}, en: {...} }` — key dạng `'section.field'`.
2. **Đánh dấu** element cần dịch bằng `data-i18n="key"`.
3. **`applyLang(lang)`** duyệt mọi `[data-i18n]`:
   - `input`/`textarea` → đổi `placeholder`; còn lại → đổi `textContent`.
   - **Bỏ qua key chưa có** (`if (!t[key]) return`) → an toàn khi thêm `data-i18n` mới mà chưa có bản dịch (giữ nguyên chữ gốc).
4. Ngôn ngữ lưu ở `localStorage('lang')`, mặc định `fr`. Nút có `data-lang-btn="fr|en"` để chuyển.
5. `Layout.astro` có `<script is:inline>` set `<html lang>` **trước paint** (chống flash nội dung FR khi đang để EN).

---

## Thêm chuỗi dịch mới

```html
<!-- markup: data-i18n trên element CHỈ chứa text (không lồng element khác) -->
<h2 data-i18n="menu.formules.title">Nos Formules</h2>
```
```js
// src/scripts/i18n.js
fr: { 'menu.formules.title': 'Nos Formules', ... }
en: { 'menu.formules.title': 'Our Set Menus', ... }
```

> ⚠️ Nếu element có con (vd chip `🌶️ <span>Épicé</span>`), đặt `data-i18n` ở **span text con**, không ở phần tử cha (vì `textContent` sẽ xoá emoji/con).

## Key của trang Menu cần bổ sung

Trang `/menu` đã gắn sẵn `data-i18n` cho nhãn tĩnh — **cần thêm các key sau vào `i18n.js`** (hiện giữ FR đến khi thêm):

| Key | FR | EN |
|---|---|---|
| `menu.subtitle` | Menu | Menu |
| `menu.title` | Découvrez nos plats | Discover our dishes |
| `menu.legend.spicy` | Épicé | Spicy |
| `menu.legend.veg` | Végétarien | Vegetarian |
| `menu.tab.formules` | Formules | Set menus |
| `menu.formules.title` | Nos Formules | Our Set Menus |
| `menu.formules.sub` | Choisissez une formule, ou composez à la carte. | Choose a set menu, or order à la carte. |
| `menu.signature` | Signature | Signature |
| `menu.tag.spicy` | Épicé | Spicy |
| `menu.tag.veg` | Végé | Veg |

> Tên/dịch **món ăn** lấy từ `src/data/menu.js` (đã có FR + EN field `translation`) → **không** cần `data-i18n`.
